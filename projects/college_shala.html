CollegeShala

CollegeShala, is as a recently funded Edu-Tech startup where I worked as a backend developer for a period of about 5 months.

While at CollegeShala, I was solely responsible for developing and maintaining the backend. The backend was made using AWS Serverless Architecture.

Students could search for their required college notes. Students could purchase credits which they could use for purchasing notes. They could keep track of their purchased notes and study them at any time.

Professors could upload notes, track their sales and redeem their earnings.

Technologies I Worked With:
  • - AWS Cognito (Authentication)
  • - AWS S3 (Storing PDF Notes)
  • - AWS DyamoDB (No-SQL Database)
  • - AWS Api Gateway (Api Routing)
  • - AWS Lambda (Computation)
  • - AWS Amplify (Hosting Frontend)
  • - AWS SES (Sending Email Notifications)
  • - Razorpay Payment
New Technologies/Features I Learnt:
  • - Using Cloud Based Services
  • - Performing Microtasks using Lambda
  • - Using Event-Driven Hooks
  • - CI/CD using AWS Amplify and Github
  • - Sending Params and Query in APIs
  • - Using DynamoDB for database
  • - AWS AIM roles and users
  • - Using AIM keys and secret ids to run scripts
  • - Integrating a Payment Gateway like Razorpay
